# SimulatioManager

### Documentation

The [SimulationManager](MAINPAGE.md) helps you managing a large number of simulations, generating input files, accessing parameters and storing key analysis data.

For more information go to the [wiki](https://github.coecis.cornell.edu/KammerResearch/ifasha/wiki) of the ifasha library.

To generate the Doxygen html documentation:

    $ doxygen configDoc


### Dependances

**Python**

Your favourite programming language.

**SQLite3**

SQLite is a software library that implements a self-contained, server-less, zero-configuration, transactional SQL database engine. SQLite is the most widely deployed database engine in the world. The source code for SQLite is in the public domain.
